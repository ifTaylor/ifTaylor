Embedded CW build

Keyer In The Rye

View the Code

Keyer In The Rye is a homebrew iambic CW keyer built around the Raspberry Pi Pico. The project explores paddle input handling, Morse timing, PWM sidetone generation, PTT control, and serial text-to-Morse playback. It started as a breadboard prototype for experimenting with the fundamentals of CW keying while keeping the design simple, inspectable, and easy to modify.

What this build covers

Paddle timing, sidetone, and key control

The firmware handles dit and dah paddle inputs, iambic squeeze behavior, adjustable WPM timing, PWM sidetone generation, onboard LED activity, PTT hang timing, and a separate logic-level CW key signal intended to drive an external interface circuit.

Current features

A working Pico-based CW trainer and keyer

  • • Dit and dah paddle inputs on GPIO
  • • PWM-generated sidetone output
  • • PTT output with hang timer behavior
  • • USB serial text-to-Morse playback
  • • BOOTSEL replay of the welcome message
Firmware behavior

USB serial Morse playback

In addition to paddle input, the Pico exposes a USB serial console. Typed text can be converted into Morse and played through the sidetone, LED, PTT output, and CW key signal. The default startup message is CQ CQ DE AF0FR, and the Pico BOOTSEL button can replay the welcome message while the firmware is running.
