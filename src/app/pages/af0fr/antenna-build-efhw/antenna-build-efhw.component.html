80m End-Fed Half-Wave Antenna

View Initial Investigation (pdf) View Xfmr Revision Investigation (pdf)

This project follows my ongoing build of a multiband 80 meter end-fed half-wave antenna. The design uses a homebrew 49:1 UNUN, a compensation coil near the feedpoint, a short counterpoise, and NanoVNA measurements to work through the tuning process. It is still in progress: the ferrite material needs to be changed, the antenna needs to be fully deployed, and the main element still needs final trimming.

Physical antenna details

80m wire, compensation coil, and counterpoise

The main radiator is expected to be around 134 feet for an 80 meter EFHW. The current build uses a compensation coil with 6 turns placed about 78 inches from the feedpoint. A short counterpoise of about 13 feet / 4 meters is being installed at deployment. The final counterpoise position and antenna shape may still shift the resonant points.

Investigation notes

The 2:14 winding cleaned up the response

The earlier 3:21 winding on Type 31 material appeared to be too aggressive for this transformer role. Type 31 is excellent for choking common-mode current, but with the added turns it may have started acting too much like a choke and not enough like a broadband impedance transformer. The current 2:14 version is closer to a conventional 49:1 EFHW transformer and produced cleaner harmonic behavior. Type 43 is still planned because it should be more appropriate for this transformer application.
