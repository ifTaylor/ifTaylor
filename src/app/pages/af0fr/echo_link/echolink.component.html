Remote Linking

UV5R + SVXLink + EchoLink Node

This project explores a low-cost DIY EchoLink simplex node built around a Baofeng UV-5R, a USB sound card, SVXLink, and a hand-built audio interface. It is also the technical foundation behind my active EchoLink node AF0FR-L, node 524009. The goal is not just getting a node on the air, but understanding the real issues behind audio levels, VOX behavior, signal routing, and why a radio build can appear to “almost work” while still sounding rough in practice.

What this build covers

Audio path, keying, and node behavior

The investigation covers the RF radio, USB sound card interface, SVXLink configuration, EchoLink registration, homemade audio coupling, receive distortion, and the tradeoffs between VOX and dedicated PTT control.

Current findings

Promising, but still being refined

  • • Receive audio is reaching SVXLink
  • • The radio can be triggered to transmit
  • • Transmitted voice is intelligible, but further filtering is being explored.
  • • Gain staging and VOX behavior are the main suspects
